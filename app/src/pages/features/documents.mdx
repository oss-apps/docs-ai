### FAQs

##### How to use DocsAI with Google Docs ?

There is no native support with google docs currently. You can export the document as `.docx` or `.pdf`
from docs and add here as file.

You can re-sync and delete documents. Send us an email at hey@docsai.app or let us know in the [Feature Requests](https://docsai.canny.io/featrue-requests) if you want to request a specific document type.
